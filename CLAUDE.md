## Project

Type: vite
Deploy: github-pages

## Structure

src/ (auth.js, main.js, render.js, sheet.js, state.js, storage.js, whatsapp.js, style.css), assets/, public/

## Rules

- Mass WhatsApp message sender - reads guest list from Google Sheets, sends via WhatsApp Web
- No framework, vanilla JS with Vite bundling
- CSS vars from settings widget, never hardcode colors
- MUST have favicon.svg + favicon.png + favicon.ico in assets/images/
